# Авторизация по токену

Авторизации пользователя по токену

***URL*** : `/auth/token/login`

***Method*** : `POST`

***Auth required*** : YES

***Permission required*** : None

## Success Responses

**Code** : `200 OK`

**Content** :

```json
{
    "auth_token": "80278cf9a4be3e6342d0592fba6aa94f0e946f74"
}
```
